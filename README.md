# Team HAMM Database Project

This is a C# Console Application that connects to a SQL Server database using Entity Framework Core.

The project includes full CRUD functionality for:
- Customers
- Products
- Orders

## Features
- View all tables
- Add customers and orders
- Update customer and order information
- Delete customers (with checks to prevent deleting linked records)

## Database
A backup of the database is included in the SQL folder:
- TeamHAMM_DB.bak
- CreateTables.sql shows the schema
